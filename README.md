# 🌾 AgroRent - Centralized Equipment Rental System

## 📌 Overview
AgroRent is a DBMS-based system that helps farmers rent agricultural equipment easily with transparent pricing and availability.

## 🚀 Features
- User registration & login
- Equipment listing by owners
- Search & filter equipment
- Booking & reservation system
- Payment tracking
- Ratings & reviews
